# Tag Report

Converts the CSV format generated by [Tag](https://github.com/gulrotkake/tag) to a HTML report.

## Usage

```
$ report --start <yyyy-MM-ddTHH:mm:ss> --end <yyyy-MM-ddTHH:mm:ss> --timezone <time zone> [--logo logo-file] data.csv
```
